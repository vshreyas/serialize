Detailed Description

Base StreamWriter abstract class. All user-implemented stream writers must inherit from this.

This provides an ostream* member and a constructor taking an open ostream object.

All derived classes must implement a (templated) `save()' function which writes data from a given variable into the stream. Whether to write the type identifer or not is to be decided by the derived class.

For classes, the user must define a `serialize()' function which serializes individual members of the class. For polymorphics, the user must first register the derived classes using REGISTER_TYPE(<writer object>,<derived class name>).
